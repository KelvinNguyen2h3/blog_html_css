Introduction

CSS, or Cascading Style Sheets, is a style sheet language used for describing the presentation of a document written in HTML. It controls the layout, colors, fonts, and overall visual appearance of web pages, allowing for the separation of content from design. CSS enables web developers to create visually engaging and responsive web designs that can adapt to different screen sizes and devices.

This is the general syntax of a Element in CSS:

selectors { Properties and Values }

For Example:

header { color: black; margin: 10px; text-align: center; font-size: 1.8em; font-weight: thin; }
Selectors

Class:

/* HTML CODE */ <div class="home"></div> /* CSS FOR IT */ .home { text-align: center; }

Id:

/* HTML CODE */ <nav id="navbar"></nav> /* CSS FOR IT */ #navbar { width: 300px; height: 100%; }

DIRECT children:

/* HTML CODE */ <section> <div></div> <div></div> <div> <div></div> /* No select it */ <div></div> /* No select it */ </div> </section> /* CSS FOR IT */ section > div { margin: 15px 0px 0px 20px; }

INDIRECT children:

/* HTML CODE */ <section> <div></div> <div></div> <div> <div></div> <div></div> </div> </section> /* CSS FOR IT */ section div { margin: 15px 0px 0px 20px; }

Double class:

/* HTML CODE */ <div class="single double"></div> <div class="single"></div> /* No select it */ <div class="single"></div> /* No select it */ /* CSS FOR IT */ .single.double { margin: 15px 0px 0px 20px; }

Fair:

/* HTML CODE */ <div></div> <nav></nav> <nav></nav> /* CSS FOR IT */ div, nav { margin: 15px 0px 0px 20px; }

INDIRECT After:

/* HTML CODE */ <div></div> <nav></nav> <nav></nav> /* CSS FOR IT */ div ~ nav { margin: 15px 0px 0px 20px; }

DIRECT After:

/* HTML CODE */ <div></div> <nav></nav> <nav></nav> /* No select it */ /* CSS FOR IT */ div + nav { margin: 15px 0px 0px 20px; }
Properties and Values

It's a pair to define the style of the selector.

For Example:

header { color: black; /* Color of the font is black */ margin: 10px; /* Margin of the box (4 direction) is 10px */ text-align: center; /* Set the text to center of the box */ font-size: 18px; /* Set the size of the font to 18px */ }

The src attribute of the img tag specifies the source URL (Uniform Resource Locator) of the image to be displayed on a web page. This attribute is mandatory for the img tag, as it tells the browser where to find the image file that should be rendered.

Responsive

Use grid-flexbox and media technique.

For Example

.body { display: flex; /* This is a flexbox */ flex-direction: column; /* box - box - box */ flex: 1 1 100px; /* It have 1 in the total - shrink ability = 1 - origin 100px */ } @media all and (max-width: 550px){ /* Only apply that CSS rule if width of the screen less than 550px */ .body{ flex-direction: column; } main{ padding: 5em 0; } }
